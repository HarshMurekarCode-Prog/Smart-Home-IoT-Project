# 🏠 Smart Home Automation System using Arduino

## 📌 Overview

This project is a Smart Home Automation System developed as a hackathon project using Arduino. It combines multiple smart home features into a single system, including fire detection, soil moisture monitoring, and home automation. The goal is to improve safety, automation, and efficient resource management.

---

## ✨ Features

* 🔥 Fire Detection System
* 🌱 Soil Moisture Monitoring
* 💡 Home Automation
* ⚡ Arduino-based Control System
* 📊 Sensor Data Processing
* 🛠 Modular Design for Future Expansion

## 🚀 How to Run

1. Install the Arduino IDE.
2. Download or clone this repository.
3. Open the required `.ino` file.
4. Select the correct Arduino board and COM port.
5. Upload the code to the Arduino.
6. Connect the required sensors and hardware.
7. Monitor the output using the Serial Monitor.
